Optional modules which can extend the functionality of Gallery Assist.
  1. Gallery Assist ImageCache (GAIC)
  2. Gallery Assist Lightboxes (GALB)
  3. Gallery Assist for Cooliris viewer (GA4CV)
  4. Gallery Assist Upload and Import Helper (GAUIH)
  5. Image FUpload for Gallery Assist (GAIFU)
  6. Gallery Assist Comments (GAC)
  7. Gallery Assist Exif (GAE)
Advanced help to each optional module is included in the respective module folder.
