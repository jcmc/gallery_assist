Gallery Assist (GA) block options Three GA block options are available.
  1. My Galleries This block display the own galleries as covers or as list. More informations coming soon.
  2. Galleries This block is similar to the My Galleries block. It shows only the galleries with enabled show in block option. More informations coming soon.
  3. Block Gallery This block presents a gallery. You can enter one or more node ID's. If several galleries have been entered, one will be shown randomly. More informations coming soon.
